Automatically remove links from posts when they are published and keep the anchor text in tact.

== Description ==

This is a plugin for those with autoblogs or outsourcing content and do not want external links on their site. It automatically strips all outgoing links.

[Plugin Homepage](http://rubensargsyan.com/wordpress-plugin-automatically-remove-links-from-posts)

[Investment provided by Ricardo Braga](http://advogadoonline.net)

== Installation ==

1. Upload the automatically-remove-links-from-posts directory (including all files within) to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
